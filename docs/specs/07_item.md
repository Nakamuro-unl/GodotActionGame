# 07 アイテム仕様書

## 概要

アイテムは消費型のリソース。宝箱やフロア上で入手し、回復・強化・特殊効果に使用する。

---

## アイテム所持

| 項目 | 仕様 |
|------|------|
| 最大所持数 | 10個 |
| 所持枠満杯時 | 拾えない（メッセージ表示） |
| 入手方法 | 宝箱、敵ドロップ、フロア上 |
| 使用 | アイテムメニューから選択（1ターン消費） |

---

## レアリティとドロップ率

| レアリティ | 宝箱出現率 | 敵ドロップ率 |
|----------|-----------|-------------|
| 普通 | 40% | 15% |
| やや稀 | 25% | 8% |
| 稀 | 10% | 3% |
| 超稀 | 3% | 1% |

※ 宝箱の中身は「知識」と「アイテム」が抽選される。知識の未獲得がない場合は100%アイテム。

---

## 罠アイテム（偽物）

一部のアイテムには偽物が存在する（ステージ3以降で出現）。

| 名前 | 見た目 | 実際の効果 |
|------|--------|-----------|
| 毒薬草 | 薬草と同じ | HP -10 |
| 混乱の水 | 知恵の水と同じ | 3ターン操作反転 |
| 呪いの巻物 | 零の巻物と同じ | 自分のMPを0にする |

偽物は使用するまで本物と区別できない。「鑑定」知識（ステージ3で入手可能）があれば判別可能。

---

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-ITM-001 | アイテムを最大10個まで所持できる |
| AC-ITM-002 | アイテムメニューから使用できる（1ターン消費） |
| AC-ITM-003 | 各アイテムが仕様通りの効果を発揮する |
| AC-ITM-004 | レアリティに基づいてドロップ率が正しい |
| AC-ITM-005 | 所持枠満杯時にアイテムを拾えない |
| AC-ITM-006 | 偽アイテムが使用するまで本物と区別できない |
| AC-ITM-007 | 鑑定知識があれば偽アイテムを判別できる |
