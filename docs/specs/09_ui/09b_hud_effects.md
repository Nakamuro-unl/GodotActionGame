# 09b UI - HUD・演出

## 概要

ゲーム画面中のHUDレイアウトと各種演出の仕様。

---

## ゲーム画面HUDレイアウト

```
+------------------------------------------+
| HP: 30/30  MP: 10/10   F:1F  Turn:0     |
| Combo: 0                                  |
+------------------------------------------+
|                                          |
|           [ダンジョンマップ]              |
|                                          |
|                                          |
+------------------------------------------+
| [1]+1  [2]-1  [3]---  [4]---  [5]---    |
| [6]---                                   |
+------------------------------------------+
| メッセージログ                            |
+------------------------------------------+
```

---

## 上部バー

| 表示項目 | 説明 |
|---------|------|
| HP | 現在HP / 最大HP（バー表示 + 数値） |
| MP | 現在MP / 最大MP（バー表示 + 数値） |
| F | 現在のフロア番号 |
| Turn | 経過ターン数 |
| Combo | 現在のコンボ数（0の時は非表示） |

---

## 技スロット表示

- スロット1-6を横に並べて表示
- 各スロットに装備中の技名を表示
- 未装備スロットは「---」
- 選択中のスロットをハイライト
- MP不足の技はグレーアウト

---

## メッセージログ

- 画面最下部に2-3行のメッセージ領域
- 直近のイベントを時系列で表示
- 例:
  - 「子狼に マイナス5 を使った! 数値: 8 → 3」
  - 「子狼を倒した! コンボ x3!」
  - 「宝箱から「乗法」を手に入れた!」

---

## ミニマップ

- ゲーム画面右上に小さなミニマップを表示
- 探索済みの部屋・通路を表示
- プレイヤー位置を点滅表示
- 「マップの欠片」使用時にフロア全体が表示される

---

## 敵の数値表示

- 敵スプライトの頭上に数値を表示
- 正の数: 白色
- 負の数（幽霊）: 紫色
- 0になった瞬間: 金色フラッシュ → 消滅アニメーション

---

## メッセージ演出

| イベント | 演出 |
|---------|------|
| ダメージ受ける | 画面を赤くフラッシュ、数値ポップアップ |
| 技命中 | 数値変化アニメーション（旧値→新値） |
| 撃破 | 金色フラッシュ + 消滅エフェクト |
| 幽霊化 | 紫色フラッシュ + 半透明化 |
| レベルアップ | 白いフラッシュ + レベルアップ表示 |
| 知識獲得 | 知識名のポップアップ + 説明表示 |
| コンボ | 「COMBO x N!」のポップアップ |

---

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-UI-002 | HUDにHP, MP, フロア, ターン数, コンボが表示される |
| AC-UI-003 | 技スロットが表示されMP不足時にグレーアウトされる |
| AC-UI-004 | メッセージログに直近のイベントが表示される |
| AC-UI-008 | ミニマップに探索済み領域が表示される |
| AC-UI-009 | 敵の数値が状態に応じた色で表示される |
| AC-UI-010 | 各イベントに対応した演出が再生される |
