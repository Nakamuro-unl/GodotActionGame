# 09a UI - 画面構成・メニュー

## 概要

ゲーム中の各画面とメニューの仕様。

---

## 画面一覧

| 画面 | 説明 |
|------|------|
| タイトル画面 | ゲーム開始、ランキング、設定 |
| ゲーム画面 | ダンジョン探索中のメイン画面 + HUD |
| メニュー画面 | 技の装備変更、知識図鑑、アイテム確認 |
| スコア画面 | ゲーム終了時のスコア表示 |
| ランキング画面 | 上位10件のスコア一覧 |

---

## タイトル画面

```
========================
   MATH MAGE
   マスメイジ
========================

> はじめから
  ランキング
  あそびかた
  せってい
```

| 項目 | 動作 |
|------|------|
| はじめから | シード値をランダム生成してゲーム開始 |
| ランキング | ランキング画面へ遷移 |
| あそびかた | 操作説明・ルール説明を表示 |
| せってい | BGM・SE音量、画面サイズ |

---

## メニュー画面（Escキーで開閉）

### タブ構成

```
[ステータス] [技] [アイテム] [知識図鑑]
```

### ステータスタブ

```
数学魔道士  Lv.5
HP: 45/45
MP: 22/22
経験値: 120/50
攻撃力: 0
防御力: 0
```

### 技タブ

```
--- 装備中 ---
[1] プラス1    (MP:0)
[2] マイナス5  (MP:2)
[3] ダブル     (MP:3)
[4] ---
[5] ---
[6] ---

--- 所持知識（装備可能） ---
  マイナス1   (MP:0)
  プラス3     (MP:1)
  プラス10    (MP:3)
```

- 知識を選択して技スロットに装備/解除
- ドラッグ&ドロップまたはカーソル選択で操作

### アイテムタブ

```
所持アイテム (5/10)
  薬草        x2
  知恵の水    x1
  マップの欠片 x1
  偶数の粉    x1
```

- アイテムを選択して使用（戦闘中は1ターン消費）
- 説明文をサブウインドウに表示

### 知識図鑑タブ

```
--- ステージ1: 石器時代 ---
[獲得] 自然数の定義  - パッシブ: 敵の数値が常に見える
[獲得] 加法         - +N で敵の数値を増やす
[????] ???          - 未発見
[????] ???          - 未発見
...
```

- 獲得済みの知識は名前・効果を表示
- 未獲得は「????」で伏せる
- 収集率をパーセントで表示

## バーチャルパッド（モバイル対応）

タッチ操作用のオンスクリーンコントロール。常時表示。

### レイアウト

```
                                    [Skill1][Skill2][Skill3]
[  ^  ]                             [Skill4][Skill5][Skill6]
[< >]    [マップ表示]
[  v  ]                             [待機]  [階段]
[移動/転換]
```

### 左側: 十字キー + モード切替

| ボタン | 動作 |
|--------|------|
| 上下左右 | 「移動」モード時: 移動（ターン消費1）|
| 上下左右 | 「転換」モード時: 方向転換（ターン消費0）|
| 移動/転換 | モード切替トグル |

### 右側: 技・アクション

| ボタン | 動作 |
|--------|------|
| Skill1-6 | 対応する技を向き方向に発動 |
| 待機 | その場で1ターン待機 |
| 階段 | 足元の階段を使用 |

## 画面解像度

| 項目 | 値 |
|------|-----|
| 基本解像度 | 640x480 px |
| タイルサイズ | 16x16 px |
| 表示タイル数 | 約20x15タイル（視界範囲） |
| スケーリング | 整数倍スケーリング対応（x1, x2, x3） |
