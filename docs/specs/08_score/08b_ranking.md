# 08b スコアシステム - ランキング・表示

## 概要

スコアをローカルに保存し、ランキングとして表示する。

---

## ランキング

| 項目 | 仕様 |
|------|------|
| 保存件数 | 上位10件 |
| 保存場所 | ローカル（user://ranking.json） |
| 記録内容 | スコア、到達フロア、撃破数、プレイ日時、シード値 |
| 表示 | タイトル画面からアクセス可能 |

### ランキングデータ構造

```json
{
  "rank": 1,
  "score": 125000,
  "floor_reached": 25,
  "enemies_defeated": 87,
  "max_combo": 12,
  "knowledge_count": 18,
  "total_turns": 1500,
  "cleared": true,
  "seed": 12345678,
  "date": "2026-04-02T15:30:00"
}
```

---

## 受け入れ条件

| ID | 条件 |
|----|------|
