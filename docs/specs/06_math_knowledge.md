# 06 数学知識体系仕様書

## 概要

数学知識はこのゲームの核となるリソース。宝箱やイベントで獲得し、戦闘技やフィールドアクションとして使用する。知識は「定義」「定理」「公式」の3カテゴリに分類される。

---

## 知識のカテゴリ

| カテゴリ | 説明 | 主な用途 |
|---------|------|---------|
| 定義 | 数学的概念の定義 | パッシブ効果・フィールドアクション |
| 定理 | 証明された命題 | 強力な戦闘技 |
| 公式 | 計算の手順 | 通常の戦闘技 |

## 知識の獲得方法

| 方法 | 説明 |
|------|------|
| 宝箱 | フロアに配置された宝箱から獲得 |
| ボス撃破 | ステージボス撃破時に確定ドロップ |
| 隠し部屋 | 特定のフィールドアクションで発見した部屋で獲得 |

### ドロップルール

- 宝箱の中身は現在ステージ以下の知識からランダム選択
- 既に獲得済みの知識は出現しない（代わりにアイテム）
- ボスは該当ステージの定理を確定ドロップ

---

## 知識の管理

- 獲得した知識は「知識図鑑」に永続的に記録される
- 戦闘技として使うには技スロット（最大6枠）に装備が必要
- フィールド効果は獲得した時点で自動的に有効
- 定義カテゴリのパッシブ効果も獲得時点で自動有効

| ID | 条件 |

| AC-KNW-008 | 既獲得の知識が宝箱から重複出現しない |
