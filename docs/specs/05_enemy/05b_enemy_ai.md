# 05b 敵キャラクター - AIパターン

## 概要

敵の行動パターン（AI）の詳細仕様。全9種のAIパターンを定義する。

---

## 共通ルール: 隣接時の停止

全AIパターン共通:
- プレイヤーと既に隣接している場合は移動しない（攻撃フェーズで攻撃）
- 1マスには1キャラのみ。他の敵がいるマスには移動できない
- 敵フェーズでは移動のたびに他の敵の最新位置を参照し、重複を防ぐ

---

## 通常追跡

```
1. プレイヤーと隣接していたら移動しない（攻撃フェーズで攻撃）
2. プレイヤーとの距離を計算
3. プレイヤーに1歩近づく（マンハッタン距離で最短方向）
4. 移動先が壁 or 他の敵 → 別方向を試す → 全方向不可なら待機
```

使用敵: 子狼、熊、サソリ、下級悪魔

---

## 直線突進

```
1. プレイヤーを視認したら直線方向に突進（壁にぶつかるまで止まらない）
2. 視認していない時は通常追跡
```

使用敵: 猪

---

## ランダム移動

```
1. 50%の確率でプレイヤー方向に移動
2. 50%の確率でランダム方向に移動
```

使用敵: 砂蛇、キメラ

---

## 鈍足追跡

```
1. 2ターンに1回だけ行動
2. 行動時は通常追跡と同じ
```

使用敵: ゴーレム

---

## 賢い追跡

```
1. A*アルゴリズムでプレイヤーへの最短経路を計算
2. 最短経路に沿って1歩移動
```

使用敵: 上位悪魔、エイリアン

---

## 巡回型

```
1. 決まった経路を巡回
2. プレイヤーが視界内（5タイル以内）に入ると通常追跡に切り替え
```

使用敵: 機械兵

---

## 逃走型

```
1. プレイヤーから離れる方向に移動
2. 3タイル以上離れたらランダム移動
```

使用敵: マッドサイエンティスト

---

## ワープ型

```
1. 3ターンに1回、ランダムな床タイルにワープ
2. ワープしないターンは通常追跡
```

使用敵: 次元虫

---

## 静止型（吸引）

```
1. 移動しない
2. 毎ターン、半径3タイル内のプレイヤーを1タイル引き寄せる
```

使用敵: ブラックホール

---

## ボスAI

```
1. 賢い追跡をベースとする
2. 以下の特殊行動を持つ:
   a. 数値が初期値の半分以下 → 攻撃力1.5倍
   b. 幽霊化時 → 数値の自然回復が+2/ターン（通常は+1）
   c. 3ターンごとに特殊技を使用（ボスごとに異なる）
```

使用敵: 原始の王、スフィンクス、魔王、計算機械、宇宙神

---

## 幽霊状態の共通挙動

- 幽霊化した敵が壁を通過できる
- 幽霊の数値は攻撃でのみ変動する（自然回復なし）
- 幽霊はプレイヤーに接触してもダメージを与えない
- AIパターン自体は幽霊化前と同じ

---

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-ENM-002 | 敵がターン制で行動パターンに従い移動する |
| AC-ENM-004 | 幽霊化した敵が壁を通過する |
| AC-ENM-005 | 幽霊の数値は攻撃でのみ変動する |
| AC-ENM-006 | 各AIパターンが仕様通りに動作する |
| AC-ENM-007 | ボスが特殊行動を実行する |
