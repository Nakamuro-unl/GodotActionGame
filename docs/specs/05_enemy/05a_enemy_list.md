# 05a 敵キャラクター - 敵一覧・数値設定

## 概要

敵はフロア内をターン制で移動し、プレイヤーに隣接すると攻撃する。頭上に数値を持ち、プレイヤーの技で数値を0にされると撃破される。

---

## 敵の基本属性

| 属性 | 説明 |
|------|------|
| 名前 | 敵の種類名 |
| 数値 | 頭上に表示される値（撃破対象） |
| 攻撃力 | プレイヤーへのダメージ量 |
| 経験値 | 撃破時に獲得する経験値 |
| 行動パターン | AI の種別（→ 05b_enemy_ai.md 参照） |
| 状態 | 通常 / 幽霊 |

## 敵の数値設定ルール

- 数値はフロア進入時にステージの範囲内でランダム決定
- ボスの数値は固定
- 数値は必ず整数
- ステージが進むにつれて「倍数」「素数」など特定の数値が出やすくなる
  - ステージ2: 2,3の倍数が多い（乗除技の練習）
  - ステージ3: 平方数が多い（関数技の練習）
  - ステージ4: 2の累乗が多い（log技の練習）

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-ENM-001 | 各敵が頭上に数値を表示する |
| AC-ENM-003 | 敵がプレイヤーに隣接時にダメージを与える |
| AC-ENM-008 | 敵の数値がステージの範囲内で生成される |
| AC-ENM-009 | ボスフロアではボス撃破まで階段が出現しない |
| AC-ENM-010 | ボス撃破時にMP回復アイテム(知恵の水)がドロップする |
| AC-ENM-011 | ボスフロアに封印ギミック+解除知識の宝箱が配置される |
