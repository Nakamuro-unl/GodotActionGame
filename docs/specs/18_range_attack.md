# 18 範囲攻撃（関数グラフ技）仕様書

## 概要

数学の関数をフィールド上にグラフとして描画し、その軌道上にいる全ての敵にヒットする範囲攻撃技。ステージ3以降で獲得可能。

---

|------|-----|

## 範囲攻撃の仕組み

### グラフ計算

```
1. プレイヤーの位置を原点(0,0)とする
2. 向きに応じて座標軸を回転:
   - 上向き: x→右, y→上
   - 下向き: x→右, y→下
   - 左向き: x→上, y→左
   - 右向き: x→上, y→右
3. 関数f(x)で各xに対するyを計算
4. (x, y)をグリッド座標に変換
5. 各グリッド座標に敵がいれば効果を適用
```

### 描画演出

- 技使用時にグラフの軌道をエフェクトで表示（0.3秒程度）
- 軌道上のタイルが一瞬光る
- 軌道上の敵にヒットエフェクト

---

## 既存システムとの統合

### CombatSystem

- SKILLSテーブルに `"type": "range"` を追加
- `"range_func"`: 関数名（linear, quadratic, sine, circle, exponential）
- `"range_damage"`: 各ヒットのダメージ量
- `"range_length"`: 射程

### 使用時の処理フロー

```
1. 技キー押下
2. 向き方向にグラフ座標を計算
3. 軌道上の全敵を取得
4. 各敵に効果適用（数値変化）
5. 軌道エフェクト描画
6. 1ターン消費
```

### 隣接不要

- 範囲攻撃は**隣接していなくても使用可能**
- 射程内に敵がいなくてもMP消費+ターン消費（空振り）
