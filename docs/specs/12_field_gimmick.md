# 12 フィールドギミック仕様書

## 概要

マップ上に配置される特殊タイル。対応する数学知識を持った状態で隣接し、決定キーで手動発動する。知識がなければ解除できない。

---

## 操作フロー

```
1. ギミックタイルに隣接する（向きをギミック方向に合わせる）
2. 決定キー（Enter / バーチャルパッド「調べる」ボタン）を押す
3. 判定:
   a. 対応する知識を持っている → ギミック解除、効果発動、メッセージ表示
   b. 知識がない → 「何かがあるが、理解できない...」表示
   c. ギミックがない方向 → 通常の「調べる」動作（階段等）
```

- ギミック解除はターン消費なし
- 解除済みのギミックは通常の床タイルに変化する
- 一度解除したギミックはフロア内で永続（戻らない）

---

## ギミックタイル一覧

| タイルID | 名前 | 出現ステージ | 必要知識 | 解除効果 |
|---------|------|------------|---------|---------|
| VOID_WALL | 「無」の壁 | 1 | 零の発見 (K-104) | 壁が消えて通路出現 |
| ICE_WALL | 氷壁 | 1 | 負の数 (K-105) | 氷が溶けて通路出現 |
| HIDDEN_PATH | 隠し通路 | 2 | 数直線 (K-106) | 見えない通路が可視化 |
| CIPHER_DOOR | 暗号扉 | 2 | 約数 (K-206) | 扉が開く |
| LOCKED_DOOR | 鍵付き扉 | 3 | 一次方程式 (K-306) | 方程式を解いて開錠 |
| GRAVITY_SW | 重力反転スイッチ | 3 | 符号反転 (K-302) | エリアの通行可否が反転 |
| WARP_ZONE | ワープゾーン | 4 | 対数 (K-404) | 制御して任意先にワープ |
| INF_CORRIDOR | 無限回廊 | 4 | 極限 (K-406) | 回廊が有限化して通過可能 |
| SPACE_WARP | 空間歪み | 5 | 位相変換 (K-505) | 空間が整理されて通過可能 |
| FINAL_DOOR | 最終扉 | 5 | 無限の定義 (K-506) | ラスボスエリア開放 |

---

## ギミック配置ルール

- 各フロアに0-2個のギミックをランダム配置
- ギミックは該当ステージ以降でのみ出現
- ギミックの先には隠し部屋（宝箱確定）または近道がある
- ボスフロア（5F, 10F, 15F, 20F, 25F）では固定配置
- 最終扉はステージ5のボスフロア（25F）に必ず配置

---

## ギミックとマップ生成の連携

### 配置手順

```
1. 通常のBSPマップ生成
2. ギミック候補位置を選定（部屋の出入口 or 壁の隣接位置）
3. ステージに応じたギミックタイルを配置
4. ギミックの先に隠し部屋を生成（壁で塞がれた小部屋）
5. 隠し部屋に宝箱を確定配置
```

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-GIM-001 | ギミックタイルがマップ上に表示される |
| AC-GIM-002 | 対応知識があれば決定キーでギミック解除できる |
| AC-GIM-003 | 知識がなければ「理解できない」メッセージが出る |
| AC-GIM-004 | 解除後にギミックタイルが床に変化する |
| AC-GIM-005 | ギミック解除でターンを消費しない |
| AC-GIM-006 | 各ステージに対応するギミックが配置される |
| AC-GIM-007 | ギミックの先に隠し部屋が生成される |
| AC-GIM-008 | 最終扉が25Fに必ず配置される |
