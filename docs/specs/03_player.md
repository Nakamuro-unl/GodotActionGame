# 03 プレイヤー仕様書

## 概要

プレイヤーは数学魔道士。ターン制でマップを移動し、数学の知識を技として使い敵と戦う。

---

## 基本ステータス

| ステータス | 初期値 | 最大値 | 説明 |
|-----------|--------|--------|------|
| HP | 30 | 999 | 体力。0でゲームオーバー |
| MP | 10 | 999 | 知識力。技の使用に消費 |
| レベル | 1 | 50 | 経験値で上昇 |

### レベルアップ

- 敵撃破時に経験値を獲得
- レベルアップ時: HP最大値 +5, MP最大値 +3, HP/MP全回復
- 必要経験値: `level * 10`（レベル1→2: 10exp、レベル2→3: 20exp）

---

## 移動

| 項目 | 仕様 |
|------|------|
| 移動方向 | 上下左右の4方向 |
| 移動単位 | 1タイル/1ターン |
| 入力 | 矢印キー or WASD |
| ターン消費 | 移動で1ターン消費 |
| 衝突 | 壁タイルには移動不可 |

### 移動の流れ

```
1. プレイヤーが方向キーを入力
2. 移動先タイルの判定:
   a. 壁 → 移動不可（ターン消費なし）
   b. 床/通路 → 移動（1ターン消費）
   c. 敵がいる → 移動不可（技使用を促す）
   d. 宝箱 → 移動 + 宝箱を開ける
   e. 階段 → 移動（決定キーで次フロアへ）
   f. 罠 → 移動 + 罠発動
3. プレイヤーターン終了 → 敵ターン開始
```

---

## 技スロット

| 項目 | 仕様 |
|------|------|
| スロット数 | 最大6枠 |
| 初期技 | 「+1」「-1」の2つ |
| 技の使用 | 数字キー 1-6 で対応スロットの技を使用 |
| 技の入れ替え | メニューから技の装備・解除が可能 |
| 技の対象 | 隣接する敵1体（方向指定） |

### 技の使用フロー

```
1. 技スロットのキーを押す（1-6）
2. 方向を指定（上下左右）
3. 指定方向に隣接する敵がいるか判定
   a. いる → 技発動（MP消費、敵の数値に効果適用）
   b. いない → キャンセル（ターン消費なし）
4. 1ターン消費 → 敵ターン開始
```

---

## その他のアクション

| アクション | キー | ターン消費 | 説明 |
|-----------|------|-----------|------|
| 待機 | Space / . | 1ターン | その場で待機 |
| アイテム使用 | I → 選択 | 1ターン | アイテムメニューを開いて使用 |
| 足元を調べる | Enter | 0ターン | 階段の使用、足元アイテム拾い |
| メニュー | Esc | 0ターン | 技の入れ替え、ステータス確認 |

---

## アイテム所持

| 項目 | 仕様 |
|------|------|
| 所持枠 | 最大10個 |
| 拾い方 | 宝箱を開ける or 床のアイテムを拾う |
| 枠が満杯 | 拾えない（メッセージ表示） |

---

## HP回復

- 自然回復: 10ターンごとにHP +1
- アイテム: 回復薬で即時回復
- レベルアップ: HP/MP全回復

---

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-PLR-001 | 4方向にタイル単位でターン制移動できる |
| AC-PLR-002 | 壁タイルには移動できない |
| AC-PLR-003 | 移動するとターンが進行する |
| AC-PLR-004 | 技スロット（最大6枠）に技を装備できる |
| AC-PLR-005 | 隣接する敵に対して技を使用できる |
| AC-PLR-006 | 技の使用でMPが消費される |
| AC-PLR-007 | MP不足時は技を使用できない |
| AC-PLR-008 | 経験値でレベルアップしステータスが上昇する |
| AC-PLR-009 | アイテムを最大10個まで所持できる |
| AC-PLR-010 | HP が 0 になるとゲームオーバーになる |
| AC-PLR-011 | 10ターンごとにHPが1回復する |
