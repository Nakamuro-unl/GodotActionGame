# 15 インベントリ仕様書

## 概要

プレイヤーは技なしでゲームを開始する。宝箱から数学知識を獲得することで技が解放され、技スロットに装備して戦闘で使用できる。インベントリ画面で獲得済み知識と技スロットを管理する。

---

## 初期状態の変更

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 初期技 | プラス1, マイナス1 | なし（全スロット空） |
| 開始部屋 | 宝箱なし | 宝箱1個確定配置（中身: 減法 K-103） |
| 技の解放 | 最初から使用可 | 知識獲得で初めて解放 |

---

## 開始部屋の宝箱

- 1Fのプレイヤー開始部屋に宝箱を1個確定配置する
- 中身は「減法」(K-103) 固定
- これにより -1 技（マイナス1）が使用可能になる
- 開始部屋の宝箱はステージに関係なく毎回配置（パーマデスで毎回1からのため）

---

## 知識と技の関係

```
宝箱を開ける
  → 知識を獲得（知識図鑑に登録）
  → 知識に対応する技が解放される
  → インベントリ画面で技スロットに装備
  → 戦闘で使用可能
```

### 技の装備ルール

- 技スロットは最大6枠
- 獲得済みの知識（公式・定理）のみ装備可能
- 定義カテゴリの知識はパッシブ効果のみ（技スロットに装備しない）
- 1つの知識を複数スロットに装備することはできない
- 宝箱で知識獲得時、空きスロットがあれば自動装備する
- スロットが満杯の場合、入れ替えるスロットを選択するダイアログが表示される
- 「入れ替えない」を選ぶと技はスロットに装備されないが、知識は獲得済みになる（後でインベントリから装備可能）
- フロア遷移時、技スロットとアイテムは維持される（リセットされない）

---

## インベントリ画面

### 開閉

- Escキー / バーチャルパッドのメニューボタンで開閉
- 表示中はゲームが一時停止（ターン消費なし）

### レイアウト

```
+------------------------------------------+
| インベントリ                    [閉じる]  |
+------------------------------------------+
| 技スロット                               |
| [1] マイナス1    [2] ---    [3] ---      |
| [4] ---          [5] ---    [6] ---      |
+------------------------------------------+
| 獲得済み知識                             |
| [装備] 減法         -N で敵の数値を減らす |
| [装備] 加法         +N で敵の数値を増やす |
| [----] 自然数の定義  パッシブ: 数値表示   |
| [----] 零の発見      フィールド効果       |
+------------------------------------------+
| アイテム (3/10)                          |
| 薬草 x2    知恵の水 x1                   |
+------------------------------------------+
```

### 操作

| 操作 | 動作 |
|------|------|
| 上下キー | 知識/アイテムリストのカーソル移動 |
| 決定キー | 選択した知識をスロットに装備（空きスロットに自動） |
| 1-6キー | 選択した知識を指定スロットに装備 |
| Escキー | インベントリを閉じる |

## 受け入れ条件

| ID | 条件 |
|----|------|
| AC-INV-001 | プレイヤーが技なしでゲーム開始する |
| AC-INV-002 | 開始部屋に宝箱が確定配置される |
| AC-INV-003 | 開始部屋の宝箱から減法(K-103)が獲得できる |
| AC-INV-004 | 知識獲得時に空きスロットへ自動装備される |
| AC-INV-005 | インベントリ画面で技の装備/解除ができる |
| AC-INV-006 | 定義カテゴリの知識は装備できない（パッシブのみ） |
| AC-INV-007 | インベントリ表示中はターンが進まない |
