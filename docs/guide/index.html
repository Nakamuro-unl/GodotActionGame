ゲーム概要

数学の知識を武器に戦う魔道士のローグライクRPG。敵の頭上に表示された数値をぴったり0にして倒す。宝箱から知識を獲得し、技やフィールドアクションとして活用しよう。

戦闘の基本

  • 敵の数値を技で変化させ、ぴったり0にすると撃破
  • 数値が負になると幽霊化（壁通過、ダメージ無効、攻撃でのみ数値変動）
  • 幽霊も技で0にしないと倒せない
  • ぴったり0で倒すとコンボ加算、幽霊化させるとコンボリセット

攻略Tips

序盤 (1F-5F): -1だけで進む。子狼(1-2)は楽に倒せる。宝箱で「繰り下がり(-5)」が出たら即装備。熊(4-8)は無理せず回避してもOK。レベルアップでHP/MP全回復するのでタイミングを計ろう。
中盤 (6F-15F): /2(ハーフ)が手に入ると世界が変わる。大きい数値の敵は /2 を連打して小さくしてから -1 で仕留める。%4(モジュロ)は4の倍数の敵に即効。偶数の粉 + /2 のコンボも強力。
終盤 (16F-25F): derivative(桁数変換)とintegral(各桁合計)が強い。100 -> derivative -> 3 -> -1x3 で撃破。identity(n/n=1)は全ての敵を1にできる最高の汎用技。ゼロ乗算はボス戦に温存。
幽霊化に注意: 数値を負にすると幽霊化。幽霊は壁を通過してくるので厄介。abs()で正に戻してから処理するか、+系の技で0に調整する。幽霊化させるとコンボがリセットされるのでスコアにも影響。
ボス攻略の定石:
  • 原始の王(10): -5 x2、または -1 x10
  • スフィンクス(24): /2 x3 -> 3 -> -1 x3、または %4 -> 0
  • 魔王(49): sqrt -> 7 -> -1 x7、または %4 -> 1 -> -1
  • 計算機械(100): derivative -> 3 -> -1 x3、または /2 連打
  • 宇宙神(256): log2 -> 8 -> /2 x3 -> 1 -> -1、またはゼロ乗算
スコアを伸ばすコツ:
  • ぴったり0撃破を連続してコンボを稼ぐ
  • 幽霊化を避ける（ペナルティ -50/回、コンボリセット）
  • ターン数を少なくする（ペナルティ -2/ターン）
  • 宝箱は全て開けて知識収集率を上げる（+200/個）
